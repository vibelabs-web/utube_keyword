## 기술적 특징

### 1. Docker 멀티 스테이지 빌드
```dockerfile
# 프론트엔드 Dockerfile
FROM node:20-alpine AS builder  # 빌드 스테이지
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build

FROM nginx:alpine  # 프로덕션 스테이지
COPY --from=builder /app/dist /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf
```

### 2. Nginx SPA 라우팅
```nginx
location / {
    try_files $uri $uri/ /index.html;
}
```

### 3. Docker Healthcheck
```yaml
healthcheck:
  test: ["CMD", "python", "-c", "import requests; requests.get('http://localhost:8000/')"]
  interval: 30s
  timeout: 10s
  retries: 3
  start_period: 40s
```

### 4. 서비스 의존성
```yaml
frontend:
  depends_on:
    backend:
      condition: service_healthy
```
