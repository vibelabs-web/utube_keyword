# Project Objectives

## 노스스타 메트릭
- 분석 도구를 통해 선정한 주제의 영상 조회수 **10,000회** 달성

## 입력 지표
- 주 1회 이상 키워드 분석 실행
- 월 4회 이상 댓글 분석 실행

## 마일스톤 목표

| 마일스톤 | 설명 | 목표 |
|----------|------|------|
| M0 | 프로젝트 셋업 | 개발 환경 구축 완료 |
| M0.5 | 계약 우선 정의 | API 계약 및 테스트 설계 완료 |
| M1 | 키워드 분석 | FEAT-1 기능 동작 |
